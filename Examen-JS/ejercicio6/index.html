Crea un script que contenga un objeto persona, implementa estos 3 métodos diferentes: introducirPersona, introduce una persona al objeto o al elemento que lo contenga, si el objeto no está vacío se alertara al usuario que está sustituyendo a la persona actual, si el objeto está vacío no se dará ningún aviso; mostrarPersona, muestra la persona creada; eliminarPersona, elimina el contenido del objeto o del elemento persona. Los atributos que debe de tener el objeto son, nombre, edad, teléfono y domicilio. (1 punto)
